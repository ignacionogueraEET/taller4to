¿Qué es la estructura condicional?

Es una estructura de toma de decisión que permite al algoritmo seguir un camino u otro según el valor de verdad (Verdadero o Falso) de una expresión lógica.

✅ Características principales

  • Permite que el algoritmo tome decisiones según una expresión lógica (Verdadero o Falso).
  • Tiene un solo punto de entrada y un solo punto de salida.
  • El bloque “Sino” es opcional.
  • Se puede anidar (una estructura dentro de otra).
  • Tanto el bloque verdadero como el falso pueden contener acciones simples u otras estructuras de control.

🛠 Elementos que podemos usar

Diagrama de Flujo

Pseudocódigo básico

Si <expresión lógica> Entonces
    Acción A;
    ...
Sino
    Acción B;
    ...
FinSi

Operadores recomendados

  • Relacionales: > < >= <= == !=
  • Lógicos: Y (AND), O (OR), NO (NOT)
  • Algebraicos: +, -, *, /, MOD

📌 Puntos importantes para recordar

  • La condición siempre se evalúa como Verdadero o Falso.
  • Si la condición es verdadera → se ejecuta el bloque “Entonces” y luego se pasa al FinSi.
  • Si es falsa → se ejecuta el bloque “Sino” (si existe) o se salta directamente al FinSi.
  • El bloque “Sino” es opcional.
  • Esta es la estructura básica para tomar decisiones en cualquier algoritmo o programa.
  • Se pueden combinar los operadores para crear condiciones más complejas.
