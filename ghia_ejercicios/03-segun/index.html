Estructura de Selección Múltiple - Según

Al finalizar esta síntesis podrás utilizar la estructura Según para resolver problemas que requieren elegir entre múltiples opciones según el valor de una variable.

🎥 Video Explicativo

https://youtu.be/DAns0C36v6s

¿Qué es la estructura Según?

Es una estructura de selección múltiple que permite cotejar el valor de una variable de control con una lista de valores posibles y ejecutar diferentes acciones según cuál coincida.

✅ Características principales

  • Evalúa el valor de una variable (numérica o caracter) y lo compara con una lista de opciones.
  • Si encuentra coincidencia, ejecuta las instrucciones correspondientes y salta al final de la estructura.
  • Si no coincide con ninguna opción, se puede usar la cláusula De Otro Modo (opcional).
  • Evita el uso excesivo de estructuras Si anidadas cuando hay muchas opciones posibles.

📊 Diagrama de Flujo

💻 Pseudocódigo

Segun <variable> Hacer
    <valor1>: instrucciones
    <valor2>, <valor3>: instrucciones
    ...
    De Otro Modo: instrucciones
FinSegun

📌 Puntos importantes para recordar

  • La variable de control puede ser numérica o de tipo caracter.
  • Se pueden agrupar varios valores en una misma opción separándolos con coma.
  • La opción De Otro Modo se ejecuta solo si ningún valor coincide.
  • Después de ejecutar las instrucciones de una opción, siempre se sale de la estructura (no continúa con las siguientes).
  • Es ideal cuando hay varias opciones discretas (menús, días de la semana, calificaciones, etc.).
