📋 Resumen Teórico

¿Qué es una estructura secuencial?

Es la forma más simple de algoritmo. Las instrucciones se ejecutan una después de la otra, en el orden en que están escritas. No hay decisiones ni repeticiones.

Características principales:

  • Se ejecuta de arriba hacia abajo.
  • Cada instrucción se realiza una sola vez.
  • Es la base de todos los algoritmos más complejos.

Elementos que podemos usar:

  • Entrada de datos: Leer / Input
  • Procesos: Asignaciones, cálculos (suma, resta, etc.)
  • Salida de datos: Escribir / Mostrar / Output

Pseudocódigo básico:

Algoritmo Saludo
    Definir nombre Como Caracter;
    
    Escribir "Ingrese su nombre";
    Leer nombre;
    Escribir "!Bienvenido ", nombre, "!";
FinAlgoritmo
